Project Title: My Goal

I am choosing to pursue an entry‑level position in Information Technology. I selected this project because it directly supports my academic and professional development. By optimizing a variety of software tools, I can set a pace where establishing myself in the IT field is possible by the end of the semester.

Personal Mission Connection

Create and refine a plan where deliverables and milestones can be met for landing an entry‑level position in the field of Information Technology.

Vision Alignment

Transition into the industry where I can apply and grow knowledge and skills in IT. Staying aligned with this trajectory will make the desired outcomes achievable.

Strategic Goal Support

Tech is broad and can feel overwhelming. Curating this personal project provides structure and prioritization for the most impactful tasks.

Professional Value

Building this project exposes me to new skills and creates artifacts I can apply to future roles.

Personal Value

It enables realistic, achievable skill‑building within the semester timeline.

Learning Outcomes

I plan to develop technical skills, gain project management experience, and deepen industry knowledge.

Hybrid Approach

I will use a Hybrid‑Iterative approach. It keeps delivery on track while allowing adaptability toward the project vision.

Microsoft Planner

I chose Microsoft Planner for its clear visuals and user‑friendly interface. I like its different board views, especially the charts tab. It supports attaching URLs to tasks; I will store source files in GitHub.
